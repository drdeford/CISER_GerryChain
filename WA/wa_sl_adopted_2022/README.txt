2022 Washington State Legislative Districts Plan

##Redistricting Data Hub (RDH) Retrieval Date
02/21/2022

##Sources
This dataset was retrieved from the Washington State Redistricting Commission at https://www.redistricting.wa.gov/final-maps.

##Processing
The RDH did not modify any of the data.

##Additional Notes
Enclosed in this zip file the shapefile file for Washington's Congressional Districts ("LEG_AMEND_FINAL.shp") and supporting files, and a block equivalency file ("LEG_AMEND_FINAL.txt").
Please note that in Washington, the same districts are used for the Washington State Senate and the Washington House of Representatives.

To keep track of a state's adopted maps, go to All About Redistricting https://redistricting.lls.edu/
