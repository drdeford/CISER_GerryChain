2022 South Carolina Congressional Plan

##Redistricting Data Hub (RDH) Retrieval Date
02/03/2022

##Sources
This dataset was retrieved via South Carolina Redistricting 2021 - Senate Judiciary Committee website: https://redistricting.scsenate.gov/planproposal.html.

##Processing
The RDH did not modify any of the data.

##Additional Notes
Enclosed in this zip file the shapefile file for South Carolina's Congressional Districts ("S865_Congress.shp") and supporting files, a state map ("House Plan 2 Senate Amendment 1 Map.pdf"), a report of statistics ("House Plan 2 Senate Amendment 1 Stats.pdf"), a report of core constituencies ("House Plan 2 Senate Amendment 1 Core Constituencies.pdf"), a partisan analysis report ("House Plan 2 Senate Amendment 1 Partisan Analysis.pdf"), a political subdivision splits report ("House Plan 2 Senate Amendment 1 Political Subdivison Splits Between Districts.pdf"), a population summary report ("House Plan 2 Senate Amendment 1 Population Summary Pop.pdf"), a voting age population summary report ("House Plan 2 Senate Amendment 1 Population Summary VAP.pdf"), and a block equivalency file ("House Plan 2 Senate Amendment 1 Block Eq.xlsx").
