2024 South Carolina State Legislative District Lower (State House) Plan

House of Representatives, Act 226 (effective 2024)

##Redistricting Data Hub (RDH) Retrieval Date
07/26/2022

##Sources
This dataset was retrieved via the South Carolina House of Representatives Redistricting 2021: https://redistricting.schouse.gov/

##Processing
The RDH created the shapefile by importing the KML data (provided by the House) into QGIS and extracted the data as a shapefile.

##Additional Notes
Enclosed in this zip file the shapefile (and supporting files) for South Carolina's State House Districts ("S1024.shp"), a block equivalency file ("S1024 BlockEq.xlsx"), a core constituencies report ("S1024 CoreConstituencies.pdf"), a demographics dataset by district ("S1024 Demographics.xlsx"), a detailed map ("S1024 Detailed Map.pdf"), a report of plan components ("S1024 PlanComponents.pdf"), a report of political subdivision splits ("S1024 Political Subdivision Splits Between Districts.pdf"), a population summary ("S1024 Population Summary.pdf"), a statewide map ("S1024 Statewide Map.pdf"), and a voting age population summary ("S1024 Voting-Age Population Summary.pdf").

To keep track of a state's adopted maps, go to All About Redistricting https://redistricting.lls.edu/
